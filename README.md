# MSE 342 Study System

Interactive exam prep for **MSE 342** — quizzes, flashcards, code exercises, short-answer practice, and mock exams.

## Live site (GitHub Pages)

**[https://demirreren.github.io/mse342-study/](https://demirreren.github.io/mse342-study/)**

Open that link in any browser; no install required. Progress (theme, streak, quiz scores) is saved in your browser’s local storage.

### Enable Pages (repo owner)

If the site does not load, enable GitHub Pages: **Repository → Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: `main` / folder: `/ (root)` → Save.**
